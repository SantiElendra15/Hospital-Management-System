Hospital Management System

Overview

Hospital Management System is a Spring Boot based web application developed to streamline hospital operations. The system provides functionalities for managing patients, doctors, appointments, and healthcare records through REST APIs.

Features

Patient Management

* Add new patients
* View patient details
* Update patient information
* Delete patient records

Doctor Management

* Add doctor details
* View doctor information
* Update doctor records
* Delete doctor records

Appointment Management

* Schedule appointments
* View appointments
* Update appointment details
* Cancel appointments

Additional Features

* REST API Based Architecture
* MySQL Database Integration
* Spring Data JPA & Hibernate

Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Maven
* REST APIs

API Endpoints

Patient APIs

* GET /patients
* POST /patients
* PUT /patients/{id}
* DELETE /patients/{id}

Appointment APIs

* GET /appointments
* POST /appointments
* PUT /appointments/{id}
* DELETE /appointments/{id}

API Testing

All REST API endpoints were tested successfully using Postman.
